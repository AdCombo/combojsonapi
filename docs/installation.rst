.. _installation:

Installation
============

.. currentmodule:: combojsonapi

Install ComboJSONAPI via ``pip`` :

.. code:: sh

    pip install combojsonapi


Development version can be downloaded from `its page at GitHub
<https://github.com/AdCombo/combojsonapi>`_. :

.. code:: sh

    git clone https://github.com/AdCombo/combojsonapi.git
    cd combojsonapi
    python3 -m venv venv
    . ./venv/bin/activate
    pip install -U pip
    python setup.py install
    pip install -r requirements-dev.txt

.. note::

    If you don't have virtualenv please install it before

    $ pip install virtualenv
